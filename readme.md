## BlackShild Planning App

This is an App which can manage user Plannings.
In this App , you can :
- You Can add new Security Agent
- You Can add new new Site
- You Can add Planning for an Agent
- You Can See all Plannings
- You Can See filter Plannings
- You receive notification about Agent cni expiration status
